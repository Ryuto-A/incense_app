# incense_app
お香のレビューを共有する Ruby on Rails アプリケーションです。

## サービス概要
「お香に特化した食べログ」をイメージした、お香レビュー共有サービスです。  
お香を嗜むユーザーが、自分の好みのお香から新たな発見や出会いを得られることを目指しています。  
ユーザーはお気に入りのお香や新しく試したお香を、ブランド・種類ごとに写真付きでレビュー投稿でき、他ユーザーの投稿やタグ検索を通じて気になる商品を探すことができます。

## このサービスへの思い・作りたい理由
私はお香を焚くことが趣味で、多いときには週に3回以上、作業時間や寝る前のリラックス時間にお香を楽しんでいます。  
特にお気に入りのブランドがあり、気分を上げたい時やリラックスしたい時など、気持ちをリフレッシュさせたい場面で欠かせない存在になっています。

初めてお香を購入した際、実際に気になっていた店舗でさまざまな香りを試しました。  
その結果、30種類以上ある香りの中で、自分が本当に気に入ったものはわずか3種類ほどでした。

さまざまな香りを試すこと自体も醍醐味ですが、  
「自分の好みの香りの系統から少しずつ広げていく」  
という出会い方ができるサービスがあっても良いのではないかと考え、このアプリを作ろうと思いました。

## ユーザー層について
ユーザー層はかなりニッチですが、老若男女を問わず、お香を好むすべてのユーザーを対象としています。  
先述した「自分の好みの香りの系統から派生させて、徐々に広げていく」体験を、多くの人に提供したいと考えています。

## サービスの利用イメージ
ユーザーはアカウント登録を行うことで、自身のお香レビューを投稿できます。  
また、他のユーザーの投稿やタグ検索を通じて、新しいお香との出会いを体験できます。  
投稿には商品ページへの外部リンクも含めることができ、気になった商品をそのまま確認・購入しやすい設計を目指しています。

## ユーザーの獲得について
将来的には、Instagram のように気になるタグをフォローできるようにし、  
該当タグに新規投稿が追加された際に通知を受け取れる仕組みを考えています。

## サービスの差別化ポイント・推しポイント
似たようなサービスとして一般的な SNS が挙げられますが、Instagram などではお香以外の投稿も多く、お香に関する情報が埋もれてしまいがちです。  

本アプリでは、お香そのもののレビューに特化することで、
- 香りの系統
- 使用感
- 写真
- 商品リンク
- タグによる検索

といった、お香選びに必要な情報へよりアクセスしやすくする点で差別化を図っています。

## コメント通知メール機能について
自分の投稿（IncenseReview）に他ユーザーがコメントした際、投稿者へ通知メールを送る機能を実装しています。

### 仕様
- コメント作成時にレビュー投稿者へ通知
- 自分の投稿に自分でコメントした場合は通知しない
- 仮メールアドレス（`@example.invalid`）には通知しない
- `deliver_later` を用いた非同期送信
- `after_create_commit` でコメント保存後に通知を発火

### 実装構成
- `Comment`  
  コメント作成後に通知サービスを呼び出す
- `CommentNotificationService`  
  通知条件の判定とメール送信処理を担当
- `CommentNotifierMailer`  
  件名・本文の生成を担当

### 開発環境での確認
開発環境では `letter_opener_web` を用いて、コメント作成時の通知メール表示を確認済みです。

### 本番運用想定
本番環境では SMTP を利用する構成を想定し、`production.rb` に Action Mailer の SMTP 設定を追加しています。  
ただし本アプリはポートフォリオ用途であり、コスト最適化の観点から、現時点では外部メールサービスの継続課金運用までは行っていません。  
必要に応じて SendGrid 等の SMTP サービスを接続できる構成にしています。
